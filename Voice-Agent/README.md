# Agente de Voz Argentino

Voice agent con acento argentino que responde sobre datos de negocio usando RAG. Pensado para soporte, seguimiento y ventas simples en pymes latinoamericanas. Stack MVP: OpenAI/Anthropic + ElevenLabs + Qdrant + FastEmbed. El diferenciador es la naturalidad local, la baja latencia y el comportamiento seguro ante incertidumbre.

## Workspace

Todo el trabajo del equipo se registra en Notion (teamspace: Agent AI). Ver `.agents/skills/voice-agent-workspace/SKILL.md` para el protocolo completo de cómo y dónde guardar cada tipo de información.
