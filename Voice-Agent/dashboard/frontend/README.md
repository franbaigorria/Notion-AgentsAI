# dashboard/frontend/

UI web del cliente. Muestra logs de llamadas, transcripciones y métricas de uso.

**Por qué existe:** El cliente necesita ver qué está pasando sin necesitar acceso técnico. Una interfaz simple con tabla de llamadas + transcripción + minutos consumidos es suficiente para el MVP.

**Estado:** Fase 7 — la tecnología frontend se define cuando llegue el momento. No especular antes de validar el producto.
