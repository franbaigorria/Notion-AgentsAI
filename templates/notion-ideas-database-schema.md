# Suggested Notion Ideas Database

Use this when you want a lightweight funnel from raw idea to active repo.

## Recommended Properties

- `Name` - Title
- `Status` - Select: Inbox, Shaping, Validating, Building, Paused, Dropped
- `Type` - Select: Voice Agent, Internal Tool, SaaS, Automation, Other
- `Problem` - Text
- `User` - Text
- `Buyer` - Text
- `MVP` - Text
- `AI Stack` - Multi-select: RAG, Voice, Memory, Tool Use, Workflow Agent, Fine-tuning
- `Moat` - Select: Data, Distribution, Workflow, Brand, Speed
- `Repo` - URL or text
- `Priority` - Select: High, Medium, Low
- `Confidence` - Number or select
- `Next Experiment` - Text
- `Last Updated` - Last edited time

## Recommended Views

- `Inbox`
- `This Week`
- `Voice / Agents`
- `Ready To Build`
- `Dropped / Learnings`

## Suggested Workflow

1. Capture raw idea quickly.
2. Reshape it into a clear problem, user, and MVP.
3. Add open questions instead of pretending certainty.
4. Decide whether to validate, build, or drop.
5. Once validated, create or link the repo.
